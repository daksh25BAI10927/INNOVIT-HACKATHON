## How to Add a New Feature

### Example: Add a Blog Page

1. **Create page file**
   ```
   src/pages/Blog.jsx
   ```

2. **Update routes in src/App.jsx**
   ```jsx
   const Blog = lazy(() => import('./pages/Blog'))
   <Route path="/blog" element={<Blog/>} />
   ```

3. **Add nav link in src/components/Navbar.jsx**
   ```jsx
   <NavLink to="/blog">Blog</NavLink>
   ```

4. **Add translation in src/i18n.js**

## SEO Meta Tags

Located in `index.html` and set dynamically with React Helmet:

```html
<meta name="description" content="Explore Indian heritage..." />
<meta property="og:title" content="Heritage & Culture Portal" />
<meta property="og:image" content="/assets/og-image.jpg" />
<meta name="theme-color" content="#7A5C3A" />
```

Each page can override with:
```jsx
<Helmet>
  <title>Custom Title</title>
  <meta name="description" content="Page description" />
</Helmet>
```
